Linux GUI/CLI Applications

  1. Battery Monitor

    2. A utility tool developed on Python3 to monitor laptop battery. It will notify the user about charging, discharging, not charging, critically low battery and any other predefined state of the battery on Linux (surely if the battery is present).

  2. LeapTime Manager

    3. Aiming to be an all-in-one, friendly to new-users, python3 based GUI backup manager for Debian/Ubuntu based systems.

  3. Linux ISO Creator

    4. Creates custom Ubuntu/Debian based Linux ISO from scratch.

  4. Simple Password Generator

    5. Very simple Python3-based GUI application to generate secure and random password. It lets an user choose:

    1. Length of password between 1 to 999.
    2. Whether to include lowercase, uppercase, digit or punctuation individually or all of them in a password.
    3. Minimum number of each cases to be included in a password.
    4. Whether to exclude some characters for each cases individually while generating a password.

  5. Theme Manager

    6. Very simple Python3-based GUI application to set different theme variants on linux.

  6. Webapp Manager

    7. Run websites as if they were apps.
